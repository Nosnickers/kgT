# 知识图谱构建项目实施计划（Apple 环境报告版）

## 项目结构

```
KG/
├── config.py                 # 配置文件（Ollama、Neo4j连接等）
├── requirements.txt          # Python依赖
├── .env.example             # 环境变量示例
├── data/
│   └── Apple_Environmental_Progress_Report_2024.md  # 数据源
├── src/
│   ├── __init__.py
│   ├── data_loader.py       # 数据加载和分块处理
│   ├── neo4j_manager.py     # Neo4j数据库操作
│   ├── entity_extractor.py  # 实体关系提取（使用Ollama + Mistral 7B）
│   ├── graph_builder.py     # 知识图谱构建核心逻辑
│   └── query_engine.py      # 图谱查询引擎
├── main.py                   # 主程序入口
└── README.md                 # 项目说明文档
```

## 实施步骤

1. **创建项目基础结构**

   * 创建目录结构

   * 初始化 Python 包

2. **配置依赖管理** (requirements.txt)

   * langchain-core

   * langchain-community

   * langchain-ollama

   * neo4j

   * python-dotenv

   * markdown

   * tqdm

3. **实现配置管理模块** (config.py)

   * Ollama 连接配置（本地服务）

   * Neo4j 连接配置

   * 模型配置（Mistral 7B）

   * 环境变量管理

4. **实现数据加载模块** (data\_loader.py)

   * 读取 Markdown 文件

   * 智能分块处理（基于章节和段落）

   * 文本预处理

5. **实现 Neo4j 管理模块** (neo4j\_manager.py)

   * 数据库连接管理

   * 创建节点（实体类型：组织、产品、材料、目标、指标等）

   * 创建关系（包含、使用、减少、实现等）

   * 批量导入优化

   * 索引管理

6. **实现实体关系提取模块** (entity\_extractor.py)

   * 使用 LangChain + Ollama 集成

   * 设计 Prompt 模板用于实体关系提取

   * 实体类型定义（Organization, Product, Material, Goal, Metric, Initiative等）

   * 关系类型定义（achieves, uses, reduces, implements, contains等）

   * 结果解析和验证

7. **实现知识图谱构建模块** (graph\_builder.py)

   * 整合数据加载、实体提取、Neo4j存储

   * 批量处理流程

   * 进度跟踪

   * 错误处理和重试机制

8. **实现查询引擎** (query\_engine.py)

   * 基础图查询（按实体类型、关系类型）

   * 路径查询（查找实体间关系链）

   * 关系网络查询（查找相关实体）

   * 统计查询（实体数量、关系分布）

9. **创建主程序入口** (main.py)

   * 命令行接口

   * 完整流程演示

   * 示例查询

10. **编写项目文档** (README.md)

    * 安装说明（Ollama、Neo4j、Python依赖）

    * 配置说明

    * 使用示例

    * 模型选择建议

## 核心功能特性

* ✅ 使用本地 Ollama + Mistral 7B 进行实体关系提取

* ✅ 支持 Markdown 文件的智能分块处理

* ✅ 针对环境报告优化的实体和关系类型

* ✅ Neo4j 图数据库持久化

* ✅ 批量处理和进度跟踪

* ✅ 灵活的查询接口

* ✅ 显存优化（适配 RTX 3060 6G）

## 预期实体类型

* **Organization**: Apple, suppliers, communities

* **Product**: MacBook Air, iPhone 15, Apple Watch

* **Material**: aluminum, cobalt, tungsten, lithium

* **Goal**: Apple 2030, carbon neutrality

* **Metric**: emissions, recycled percentage, water usage

* **Initiative**: Power for Impact, Grid Forecast

* **Location**: Nepal, Colombia, India, Brazil

## 预期关系类型

* **ACHIEVES**: Organization → Goal

* **USES**: Product → Material

* **REDUCES**: Initiative → Metric

* **CONTAINS**: Product → Material

* **IMPLEMENTS**: Organization → Initiative

* **LOCATED\_IN**: Initiative → Location
