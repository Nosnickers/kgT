# 虚拟环境设置和使用计划

## 1. 创建虚拟环境
```bash
python -m venv venv
```

## 2. 激活虚拟环境（Windows PowerShell）
```bash
.\venv\Scripts\Activate.ps1
```

## 3. 优化依赖版本约束
- 修改 requirements.txt，将部分严格版本（`==`）改为宽松版本（`>=`）
- 或先尝试安装，根据错误信息调整

## 4. 安装依赖
```bash
pip install -r requirements.txt
```

## 5. 更新文档
- 在 QUICKSTART.md 中添加虚拟环境使用说明
- 说明如何激活/停用虚拟环境

## 6. 验证和使用
- 运行 `python main.py --help` 验证安装
- 使用 `python main.py --build` 构建知识图谱

## 虚拟环境使用说明：
- 每次打开新终端需要激活：`.\venv\Scripts\Activate.ps1`
- 停用虚拟环境：`deactivate`
- 激活后直接使用 `python` 命令即可
