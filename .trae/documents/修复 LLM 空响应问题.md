## 问题分析
日志显示 LLM 返回了空响应，导致实体和关系提取数量为 0。可能原因：
1. **Stop tokens 导致提前终止**：当 `deep_thought_mode=false` 时，设置了 `stop=["<think>", "</think>"]`，可能导致 LLM 一开始输出 `<think>` 标签就立即停止，产生空内容。
2. **提示词过于严格**：当前提示词包含大量“不要做”的指令，可能导致 LLM 过于谨慎，拒绝生成任何内容。
3. **响应清理问题**：`clean_json_response` 方法可能将无效响应转换为空 JSON `{}`。
