# 知识图谱原型实施计划（从零开始）

## 📋 项目目标

使用 LangChain + Ollama (Mistral 7B) + Neo4j，从 Apple 环境报告中提取实体和关系，构建知识图谱。

## 🗂️ 项目结构

```
KG/
├── config.py                 # 配置管理
├── requirements.txt          # Python 依赖
├── .env                      # 环境变量（不提交到 git）
├── .env.example             # 环境变量示例
├── data/
│   └── Apple_Environmental_Progress_Report_2024.md
├── src/
│   ├── __init__.py
│   ├── data_loader.py       # 数据加载和分块
│   ├── neo4j_manager.py     # Neo4j 数据库操作
│   ├── entity_extractor.py  # 实体关系提取
│   └── graph_builder.py     # 图谱构建流程
├── main.py                   # 主程序
└── README.md                 # 使用说明
```

## 🎯 核心功能特性

* ✅ 使用本地 Ollama + Mistral 7B

* ✅ 智能分块处理 Markdown

* ✅ 针对 Apple 环境报告优化的实体/关系类型

* ✅ Neo4j 持久化存储

* ✅ 批量处理和进度跟踪

* ✅ 适配 RTX 3060 6G 显存

## 📦 依赖工具

* **Ollama** - 本地 LLM 服务

* **Neo4j** - 图数据库（Desktop 或 Docker）

* **Python 3.10+** - 编程语言

## 🚀 快速开始流程

1. 安装 Ollama 并拉取 Mistral 7B 模型
2. 安装 Neo4j 并启动服务
3. 创建 Python 虚拟环境
4. 安装依赖（pip install -r requirements.txt）
5. 配置 .env 文件
6. 运行 main.py
